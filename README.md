#Spring Boot - Activiti

##Description：
该项目是关于 Spring Boot 整合 Activiti 框架的一个代码笔记，只用于学习使用，非商业性质。所有的笔记都在Test文件夹下，
因为都是使用 Junit 进行测试使用的。

###开发环境：
- IDEA：Spring Tool Suite 4
- JDK：1.8 version
- SpringBoot：2.1.3.RELEASE
- Activiti：6.0.0
- Lombok：1.18.6

##Note：
- Activit - Basis
    - 流程部署
    - 流程定义
    - 流程实例
    - 执行实例
    - 办理任务
    - Activiti Service

- Activit - Manage
    - 管理流程定义
    - 管理流程历史
    - 管理流程实例
    - 管理流程变量

- Activit - Sequence
    - 连线变量

- Activit - Gateway
    - 排他网关(ExclusiveGateway)
    - 并行网管(ParallelGateway)
