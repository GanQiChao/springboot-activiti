# Spring Boot - Activiti

## Description：
该项目是关于 Spring Boot 整合 Activiti 框架的一个代码笔记，只用于学习使用，非商业性质。所有的笔记都在Test文件夹下，
因为都是使用 Junit 进行测试使用的。

### 开发环境：

- Spring Tool Suite：Spring Tool Suite 4 版本
- JDK：1.8 版本
- SpringBoot：2.1.3.RELEASE 版本
- Activiti：6.0.0 版本
- Lombok：1.18.6 版本
- MyBatis：2.0.0 版本
- Mybatis - Plus：3.1.0 版本
- MySQL：8.0.15 版本
